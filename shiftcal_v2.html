📱 Import on iPhone:
1. Tap an export button above → file downloads
2. Tap the downloaded .ics file
3. iOS asks to open in Calendar → tap Add All
4. Shifts appear with built-in alarm alerts
5. Re-export monthly when you open this app
🔔 Each event has alarm reminders at your configured times (set in Settings → Alarms per Shift). Alarms ring even when iPhone is silenced — enable Calendar notifications in iPhone Settings → Notifications → Calendar.
Delete from iPhone Calendar
⚠️ Do this FIRST if Calendar shows wrong events:
1. Open Calendar app on iPhone
2. Tap Calendars at the bottom
3. Tap ⓘ next to ShiftCal (or My Shift Schedule if present)
4. Scroll down → tap Delete Calendar
5. Repeat for the other calendar name if it also exists
6. Come back here and export fresh — all events will show correctly
