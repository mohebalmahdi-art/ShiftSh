⚡ Shortcuts Auto-Sync Setup
This script runs every night at midnight via the Shortcuts app. It:
✓ Checks tomorrow's shift from your pattern
✓ Creates up to 3 alarms only if not already set
✓ Deletes yesterday's passed alarms automatically
✓ Adds tomorrow's shift to iPhone Calendar

Create the Shortcut

Step 03 — Automate it
Run at Midnight Every Night
1. Go to Automation tab in Shortcuts
2. Tap + → Time of Day
3. Set time to 12:00 AM, repeat Daily
4. Choose "ShiftCal Sync" shortcut
5. Turn off "Ask Before Running" → Done

It now runs silently every night. ✓
Smart alarm logic:
• Checks existing alarms before creating new ones
• Uses alarm labels like "ShiftCal-Morning-1" to identify its own alarms
• Deletes previous day's ShiftCal alarms after they've passed
• Creates 1, 2 or 3 alarms per your setting, each 5 min apart
